Extend methods

Example

julia> module Foo
           hello() = "Hello!"
       endMain.Foo
julia> module Bar
           using ..Foo
           Foo.hello(name) = "Hello, $name."
           println(Foo.hello())
       endHello!
Main.Bar
julia> using .Bar
julia> using .Foo
julia> methods(Foo.hello)# 2 methods for generic function "hello" from Main.Foo:
 [1] hello()
     @ REPL[1]:2
 [2] hello(name)
     @ Main.Bar REPL[2]:3

Example2

julia> module Foo
           hello() = "Hello!"
       endMain.Foo
julia> module Bar
           using ..Foo
           hello = Foo.hello
           Bar.hello(name) = "Hello, $name."
       endMain.Bar
julia> using .Bar
julia> methods(Bar.hello)# 2 methods for generic function "hello" from Main.Foo:

     @ REPL[1]:2

     @ Main.Bar REPL[2]:4
